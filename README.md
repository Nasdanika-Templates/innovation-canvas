This is a template repository to create Innovation Canvases. 
It is based on [Nasdanika Draw.io Site](https://nasdanika-templates.github.io/drawio-site/) template.
Follow instructions in the template to fill-out the canvas by adding nodes for canvas elements - problems, opportunities,
"How might we" statements to "The Art of Possible" container, ...

[TOC levels=6]

## Desirability

### Problem / Opportunity

- What is the unmet need?
- Who experiences it?
- Why does it matter?

### The Art of Possible (a.k.a. How Might We… (HMW))

A dedicated block with 1–3 crafted HMW questions  

HMW formula:  **How might we + <action> + for + <user> + so that + <outcome>**  
  
### Customers / Users

- Personas  
- Pain points  
- Wants/Needs
- Context

### Value Proposition

- What value do we create?
- What changes for the user?

## Solution

- Early concept  
- Prototype direction  
- Key assumptions to test

## Feasibility

- Technical constraints  
- Capabilities  
- Dependencies

## Viability

- Business model  
- Costs  
- Risks  
- Success metrics
